## How to build

### Requirements

- [Node.js 18.12.1 + NPM](https://nodejs.org/en/download/)
- Node-gyp

```sh
npm i node-gyp
```

### Buiding

Configure:

```sh
npx node-gyp configure
```

Build:

```sh
npx node-gyp bild
```
